üìã Installation

Add weather to your site in 2 simple steps:

Step 1: Add this HTML where you want the widget

<div id="cloudvibes-widget" data-city="Paris" data-theme="light"></div>

Step 2: Add this script before closing </body> tag

<script src="https://cloudvibes.org/widget.js"></script>

That's it! The widget will automatically load with current weather data.

Multiple Widgets

You can add multiple widgets to the same page with different cities:

<div id="cloudvibes-widget-2" data-city="London"></div>

Dynamic Loading

Load widgets dynamically with JavaScript:

// Create widget element
const widget = document.createElement('div');
widget.id = 'cloudvibes-widget-dynamic';
widget.setAttribute('data-city', 'Dubai');
widget.setAttribute('data-theme', 'dark');
document.body.appendChild(widget);

// Initialize widget
CloudVibesWidget.init(widget);
